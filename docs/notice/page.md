## About This Project

This project is an independent, community-driven effort to bring Linux support to Universal Audio Apollo Thunderbolt audio interfaces. It is not affiliated with, endorsed by, or sponsored by Universal Audio.

This work exists because we love Universal Audio products. Apollo interfaces are exceptional pieces of audio hardware, and we believe their owners should be able to use them on any operating system they choose.

## Intellectual Property

All code in this repository is original work based on clean-room reverse engineering of hardware interfaces — observing publicly accessible register behavior, wire protocols, and device responses. No proprietary source code, firmware, or copyrighted materials from Universal Audio are included.

We respect Universal Audio's intellectual property. If any content in this repository is believed to infringe on UA's rights, we welcome direct communication and will promptly make adjustments.
